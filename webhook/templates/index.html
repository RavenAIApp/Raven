What we are using

ChatGPT

ChatGPT is an open-source natural language processing (NLP) library that enables developers to easily create conversational interfaces. It is a transformer-based language model based on the GPT-4 model that is trained on conversation data. It can generate responses to user inputs, allowing for natural language conversations between users and bots.

Weaviate

Weaviate is a knowledge graph engine that helps developers understand and interpret user input, extract information from input data, and execute queries against a knowledge graph. It is built on the graph theory, and uses an object-oriented programming model to store, query, and explore data.

With Weaviate, developers can develop applications that use natural language processing (NLP) to understand user intent, retrieve and store data, and query the knowledge graph.

Messenger bot

A messenger bot is a type of chatbot that allows users to interact with a business or service via a messaging app like Facebook Messenger, Slack, or Telegram. The bot is programmed to understand and respond to specific commands and queries from the user, helping to quickly answer questions, provide information, and even complete transactions.

The bots are powered by Artificial Intelligence (AI) and Natural Language Processing (NLP) technologies, enabling them to understand natural language and context so they can accurately provide answers to user queries. Messenger bots can be used for customer service, marketing, sales, and operations, making them an essential tool for businesses to stay connected with their customers.
